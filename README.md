# ts-rpg-inventory-kata

> Simple RPG Management Inventory for training .ts and Jest!

## Para que este projeto foi feito?

Este projeto nasceu com um objetivo claro: **treinamento e aprimoramento técnico**. Ele atua como um *Kata* (exercício prático de código) focado em:

* **Programação Orientada a Objetos (POO):** Aplicar fortemente os conceitos de POO utilizando **TypeScript**.
* **Testes Automatizados:** Melhorar a prática de testes unitários utilizando o **Jest**.

## O que é o projeto?

É um sistema simples de armazenamento e gerenciamento de itens de RPG, rodando inteiramente em memória (através de manipulação de Arrays). 

Apesar de simples, ele implementa regras de negócio comuns em jogos, como:
* **Sistema de Slots:** Limitação de espaço no inventário.
* **Sistema de Peso:** Controle da carga total suportada.
* **Stack de Itens:** Capacidade de agrupar itens do mesmo tipo (dependendo da configuração do item).

## Tecnologias

* TypeScript
* Jest
